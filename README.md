# Personal Finance Tracker

This project is a command-line application for tracking personal expenses using Python and SQLite. Users can add transactions, view all expenses, and generate spending summaries by category.

## Features

- Add expense records
- View all transactions
- Calculate total spending
- Show spending by category
- Display the highest expense
- Store transaction data in SQLite

## Tech Used

- Python
- SQLite
- SQL

## How to Run

1. Open a terminal in the project folder
2. Run:
   python main.py

## Example Menu

1. Add Expense
2. View Expenses
3. Show Total Spending
4. Show Spending by Category
5. Show Highest Expense
6. Exit
